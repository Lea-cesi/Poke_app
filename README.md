# Poke_app

Il permet de connaître le pokémon qui vous représente à partir de votre prénom.
Ce code est dépendant et fait appelle à une requête API sur le pokeapi.co sur le port 443 en https et il écoute sur le port 8080 en http.
Il est dans le language Golang.

Pour faire fonctionner l'appli, il vous faut installer le language GO et vous pourrez lancer le main.go avec un "go run" dans un cmd sur le repertoire contenant les fichiers.
URL d'accès : http://localhost:8080
